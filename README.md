### Переменные окружения
Для настройки переменных окружения необходимо создать файл `.env`.  
Можно воспользоваться командой:

```bash
make env
```

### Работа с сервисом

#### Запуск сервера

```bash
make run
```

#### Создать базу данных
```bash
make create_db
```

#### Применить миграции
```bash
make migrate-up
```

#### Откатить миграции
```bash
make migrate-down
```

#### Запустить тесты
```bash
make test
```

#### Форматирование кода
```bash
make fmt
```

#### Проверка кода на потенциальные ошибки
```bash
make vet
```

#### Обновление зависимости
```bash
make tidy
```
